Seletores CSS

Bem-vindo!

Seletores definem em quais elementos nós vamos aplicar CSS:

  • Seletor por Tipo: Busca elementos por uma tag HTML específica.
  • Seletor por ID (#): Busca elementos através do atributo id.
  • Seletor por Classe (.): Busca elementos através do atributo class.
  • Seletor Universal (*): Seleciona todos os elementos HTML.
  • Seletor de Atributo ([atrib]): Seleciona elementos que possuem um atributo específicos em nosso documento HTML e consegue buscar atributos com um valor específico.
