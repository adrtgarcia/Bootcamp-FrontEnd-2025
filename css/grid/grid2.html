  • É possível usar a unidade FR para determinar o tamanho das colunas e linhas.

Áreas do Grid

  • É possível usar o shorthand GRID para definir as posições das linhas e colunas.
  • É possível usar o shorthand GRID-TEMPLATE para definir as propriedades vistas anteriormente de uma só vez.
  • É possível nomear as áreas do grid e posicionar os itens dentro de áreas específicas:
    grid-template-areas: "main main . sidebar"

Espaçamento Entre as Faixas do Grid

  • COLUMN-GAP: distância entre colunas
  • ROW-GAP: distância entre linhas
  • GAP: shorthand (gap: row column)
