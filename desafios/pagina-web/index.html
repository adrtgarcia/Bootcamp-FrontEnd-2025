Aula 01: Introdução ao HTML na Prática

(Voltar para o índice)

Para a estruturação básica do HTML, contamos com algumas TAGs para formatação de texto, como:

  • strong ou b para colocar o texto em negrito;
  • i para escrever em itálico;
  • u para sublinhar um texto.

Além disso, os títulos e subtítulos são indicados pelas TAGs h1, h2, ..., h6, enquanto os parágrafos são sinalizados por p. Para citações, é possível utilizar blockquote para uma citação longa, ou ainda q para uma citação mais curta. Por fim, a TAG br é utilizada para quebra de linha.

Para a criação de listas, temos as TAGs ol para listas enumeradas, ul para listas com bullet points, e li para listar os itens.

Já para a navegação com links é possível por meio da TAG a. Podemos adicionar links externos, links para outra página do mesmo projeto, ou ainda ancorar elementos dentro de uma página.

Por fim, temos alguns inputs básicos, sendo eles text para entradas do teclado, number para tratar números, e ainda color para selecionar cores.

Aula 03: Estruturando seu HTML + Formatações

(Voltar para o índice)

Para a formatação de textos em uma página em HTML, temos diversas TAGs:

  • i: Texto em itálico
  • strong: Texto em negrito
  • u: Texto sublinhado
  • mark: Texto destacado
  • sup: Texto superescrito
  • sub: Texto subescrito

Para citações, podemos optar por q para citações breves, como Lorem ipsum dolor sit amet consectetur adipisicing elit, ou ainda, blockquote para citações mais longas:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Suscipit repellat maxime ex nemo beatae dolor quaerat nisi deserunt incidunt doloribus earum, iste, amet repellendus, praesentium soluta aut adipisci modi in?

Já para a estruturação de uma página web, podemos utilizar embeds e iframes:

Um Embed permite a inserção de um conteúdo externo que não é HTML, normalmente mídia ou aplicações.

Por sua vez, um Iframe insere outra página HTML dentro da página atual.


Aula 04: Trabalhando com Mídias Utilizando HTML

(Voltar para o índice)

Para adicionar diferentes tipos de mídias em uma página HTML, é possível contar com TAGs específicas para cada tipo, como podemos ver a seguir.

A TAG img exibe imagens, sejam elas PNG, JPEG, JPG, GIF...

Já a TAG audio insere um player de áudio no navegador.

Apesar da TAG video funciona como a TAG audio, ela permite múltiplos source. Por sua vez, é possível adicionar legensas e faixas de texto para áudios e vídeos por meio da TAG track, utilizando um arquivo VTT.

Por fim, o iframe incorpora o conteúdo de outra página.


Aula 05: Criando Tabelas com HTML

(Voltar para o índice)

As tabelas permitem formatar as informações de forma tabular, sendo suas principais TAGs:

  • table para definir uma tabela;
  • caption para adicionar título/legenda à tabela;
  • tr para adicionar uma nova linha;
  • td para adicionar uma nova coluna, ou célula de dado;
  • th para adicionar uma célula de título
  • thead indica o cabeçalho da tabela;
  • tbody indica o corpo da tabela;
  • tfoot indica o rodapé da tabela.

Aula 06: Entedendo HTML Semântico

(Voltar para o índice)

O HTML Semântico diz respeito ao uso de TAGs que têm significado para estruturar o conteúdo de forma lógica e clara, tanto para humanos quanto para máquinas (navegadores, leitores de tela, mecanismos de busca). Assim, ao invés de usar apenas div e span para tudo, usamos header, nav, main, article, section, dentre outras TAGs, indicando sua função e/ou contexto.

Com isso, ganhamos como benefícios a acessibilidade, a manutenção do código, e o melhor entendimento do conteúdo das páginas pelos SEOs, além de separar a estilização via CSS da formatação do HTML.
